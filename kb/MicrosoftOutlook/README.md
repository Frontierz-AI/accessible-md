# Microsoft Outlook for Windows

Accessibility reference for Classic Outlook and New Outlook on Windows.

## What's covered

- **Screen readers:** JAWS, NVDA, Narrator (per-reader behavior differences, mode switching, auto-read behavior)
- **Low vision tools:** Windows Magnifier, high contrast themes, display scaling, dark mode, Immersive Reader
- **Keyboard shortcuts:** 60+ shortcuts across both Outlook versions, organized by scope
- **Interface layout:** Region maps and F6 navigation order for both versions
- **Settings paths:** Step-by-step paths to every accessibility-relevant setting
- **Task procedures:** 18 task-by-task guides (read, compose, reply, attachments, calendar, contacts, rules, signatures, spell check, Copilot, and more)
- **Calendar grid navigation:** Detailed guide for the Classic Outlook calendar, a known pain point for people who use screen readers
- **Troubleshooting:** Common problems and fixes for screen reader focus, pop-up dialogs, and version differences

## Verified against

- New Outlook for Windows (2025-2026 builds)
- Classic Outlook (Microsoft 365, 2025-2026)
- JAWS 2024-2025
- NVDA 2024.x
- Windows Narrator (Windows 11)
- Windows Magnifier, Contrast Themes

## Known gaps

- Outlook for Mac, iOS, and Android are not covered.
- VoiceOver and TalkBack are not covered.
- Some New Outlook features are still evolving and may change with updates.
